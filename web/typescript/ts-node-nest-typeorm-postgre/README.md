# ts-node-nest-typeorm-postgre notes

source: https://medium.com/simform-engineering/nestjs-and-postgresql-a-crud-tutorial-32aa78778752

## prepare packages
1. `npm i -g @nestjs/cli`
1. `nest new project-name`
1. `npm install --save @nestjs/typeorm typeorm pg class-validator class-transformer`
1. `npm uninstall prettier` and remove everything related to prettier
1. `npm run start:dev`
1. `nest g res user` -> to create CRUD for users

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
